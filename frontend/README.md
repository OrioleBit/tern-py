
# Order App Front

This is a frontend for order app management.

## Getting Started

First, run the development server:

    npm run dev

## Project Structure

    root/
    ├── app/
    ├── components/
    ├── features/
    │    └── auth/
    │        ├── components/
    │        ├── hooks/
    │        ├── services/
    │        └── types/
    ├── hooks/
    ├── services/
    ├── store/
    ├── types/
    └── utils/

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
